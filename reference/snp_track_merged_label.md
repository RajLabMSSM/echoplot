# Add SNP labels

Add SNP labels for Manhattan plot. Support function for
[plot_locus](https://rajlabmssm.github.io/echoplot/reference/plot_locus.md).

## Usage

``` r
snp_track_merged_label(
  snp_plot,
  yvar = "PP",
  genomic_units = "Mb",
  show.legend = TRUE,
  max.overlaps = 30,
  verbose = TRUE
)
```

## Arguments

- genomic_units:

  Which genomic units to return window limits in.

- verbose:

  Print messages.
