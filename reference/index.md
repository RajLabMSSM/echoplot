# Package index

## Locus Plotting

Main functions for creating multi-track locus plots.

- [`plot_locus()`](https://rajlabmssm.github.io/echoplot/reference/plot_locus.md)
  : Generate a locus plot
- [`plot_locus_multi()`](https://rajlabmssm.github.io/echoplot/reference/plot_locus_multi.md)
  : Plot multi-GWAS/QTL or multi-ancestry fine-mapping results

## Individual Tracks

Functions for generating individual plot tracks.

- [`snp_track_merged()`](https://rajlabmssm.github.io/echoplot/reference/snp_track_merged.md)
  : SNP track merged
- [`transcript_model_track()`](https://rajlabmssm.github.io/echoplot/reference/transcript_model_track.md)
  : Plot transcripts
- [`ROADMAP_plot()`](https://rajlabmssm.github.io/echoplot/reference/ROADMAP_plot.md)
  : Plot annotation peaks: ROADMAP
- [`XGR_plot()`](https://rajlabmssm.github.io/echoplot/reference/XGR_plot.md)
  : Plot annotation peaks: XGR
- [`XGR_plot_peaks()`](https://rajlabmssm.github.io/echoplot/reference/XGR_plot_peaks.md)
  : Plot XGR peaks
- [`dot_summary_plot()`](https://rajlabmssm.github.io/echoplot/reference/dot_summary_plot.md)
  : Dot summary plot

## Utilities

Helper and configuration functions.

- [`get_palettes()`](https://rajlabmssm.github.io/echoplot/reference/get_palettes.md)
  : Get palettes
- [`get_tx_biotypes()`](https://rajlabmssm.github.io/echoplot/reference/get_tx_biotypes.md)
  : Get transcript biotypes
- [`list_namespace()`](https://rajlabmssm.github.io/echoplot/reference/list_namespace.md)
  : List namespace
